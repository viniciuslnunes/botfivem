# Bot Discord Node.js para FiveM Torcidas

## Como usar

1. Renomeie o arquivo `.env` e coloque seu token do bot:
   
   BOT_TOKEN=SEU_TOKEN_AQUI

2. Instale as dependências (já feito):
   
   npm install

3. Inicie o bot:
   
   node index.js

## Comando disponível
- !ping → O bot responde "Pong!"
